# List of emojis

[![](//upload.wikimedia.org/wikipedia/commons/thumb/e/ef/Emoji_u263a.svg/40px-Emoji_u263a.svg.png)](/wiki/File:Emoji_u263a.svg "File:Emoji u263a.svg")

You may need [rendering support](/wiki/Help:Special_characters "Help:Special characters") to display the [Unicode emoticons or emoji](/wiki/Emoji#In_Unicode "Emoji") in this article correctly.

Unicode 17.0 specifies a total of 3,953 [emoji](/wiki/Emoji "Emoji") using 1,438 characters spread across 24 blocks, of which 26 are [Regional indicator symbols](/wiki/Regional_indicator_symbol "Regional indicator symbol") that combine in pairs to form flag emoji, and twelve (âŸ¨#âŸ©, âŸ¨*âŸ© and âŸ¨0âŸ©â€“âŸ¨9âŸ©) are base characters for [keycap](/wiki/Keycap "Keycap") emoji sequences.[1][2][3]

45 code points in the [Dingbats](/wiki/Dingbat#Unicode "Dingbat") block are considered emoji. All of the code points in the [Emoticons](/wiki/Emoticons_\(Unicode_block\) "Emoticons \(Unicode block\)") block are considered emoji. 92 code points in the [Miscellaneous Symbols](/wiki/Miscellaneous_Symbols "Miscellaneous Symbols") block are considered emoji. 652 code points in the [Miscellaneous Symbols and Pictographs](/wiki/Miscellaneous_Symbols_and_Pictographs "Miscellaneous Symbols and Pictographs") block are considered emoji. 242 code points in the [Supplemental Symbols and Pictographs](/wiki/Supplemental_Symbols_and_Pictographs "Supplemental Symbols and Pictographs") block are considered emoji. All of the code points in the [Symbols and Pictographs Extended-A](/wiki/Symbols_and_Pictographs_Extended-A "Symbols and Pictographs Extended-A") block are considered emoji. 106 code points in the [Transport and Map Symbols](/wiki/Transport_and_Map_Symbols "Transport and Map Symbols") block are considered emoji.

## Additions

[[edit](/w/index.php?title=List_of_emojis&action=edit&section=6 "Edit section: Additions")]

Some vendors, most notably Microsoft, Samsung and [HTC](/wiki/HTC "HTC"), add emoji presentation to some other existing Unicode characters or coin their own ZWJ sequences.

The open source projects Emojidex and Emojitwo are trying to cover all of these extensions established by major vendors.
